Description:
This is shopping list based application. It requires user to put his name in username 
column and enter register name button. Once successfully registering, shoppingList page 
shows up and with an option to add or remove items by entering the item, and clicking add 
or delete button. A link to logout from the session is added to shoppingList page which
will bring back to register page.
